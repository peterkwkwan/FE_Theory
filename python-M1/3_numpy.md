# Numpy

- Linear algebra library for Python
- A.K.A Numberical Python

#### Why use it?

- Many built-in functions and capabilities
- Import aspects of Numpy
  - arrays
  - vectors
  - matrices
  - number generation

#### Numpy arrays

- Numpy arrays come in 2 flavors: vectors & matrices
- Vectors: 1-d arrays
- Matrices: 2-d arrays
